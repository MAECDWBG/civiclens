# CivicLens 🌿
### AI-Powered Community Issue Reporting Platform

CivicLens enables citizens to report, verify, and track local infrastructure 
problems using AI. Built for the Community Hero hackathon challenge.

## 🚀 Live Demo
[Click here to view the app](https://mayukh-civiclens.netlify.app/)

## ✨ Features
- 📍 Report issues with AI-powered categorization & severity scoring
- ⚡ Gemini 2.0 Flash analyzes each report instantly
- 📄 Auto-generates official complaint letters to government departments
- 📊 Community dashboard with impact charts & AI insights
- 👥 Community upvoting & verification system
- 🏆 Gamification — points, badges & leaderboard

## 🛠 Tech Stack
- Vanilla JS / HTML / CSS (frontend)
- React + Tailwind (source in CivicLens.jsx)
- Google Gemini 2.0 Flash API (AI features)
- Netlify (deployment)

## 🤖 Google Technologies Used
- Gemini 2.0 Flash — issue classification, complaint drafting, pattern insights
- Google AI Studio — development environment

## ▶️ Run Locally
Open `CivicLens.html` in any browser. Add your Gemini API key at line 4.
